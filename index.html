HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML means Hypertext Markup Language
  • HTML is made up of multiple element that each play a certain role and provide structure to web pages
  • Some elements used in HTML include html, head, title, meta, !DOCTYPE, html, and title
  • Some elements are seen by users while others are not. The ones not seen by users help to render the page correctly
  • Parent elements have other elements inside of them called child elements
  • We can give structure and assign roles to content by using semantic HTML elements

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS stands for Cascading Style Sheets
  • CSS creates the style of the webpage and makes the page more appealing
  • Style sheets must be linked to the HTML file in order for it to work on the page
  • There are 3 ways to add CSS which include internal CSS style sheet, external CSS style sheet, and inline CSS
  • A declaration contains two main parts, the property and the value attached to that property

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -A: adds changes made to the file to the staging area
  • git commit -m “add comment here”: commits changes to repository
  • git pull origin main: makes sure the branch that is most current is up to date
  • git push origin “name of branch”: push all changes
  • git checkout “name of branch”: moves you to different branches
  • git pull origin main: pulls the remote main branch to the local main branch

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript allows users to be able to interact with applications
  • JavaScript also uses variables which are able to store data types as values
  • JavaScript will usually execute code from top to bottom
  • Conditional statements can interrupt control flow by using if statements
  • If statements create conditions for the computer to evaluate and if it is true it will execute the code in the curly brackets, if it isn’t true then it will interrupt the control flow
  • Arrays store similar data in one variable
  • For loops help to preform certain tasks multiple times
  • Functions: blocks of code that can be reused and preform a certain task
  • In order to declare a function you need to use the word function and then give the function a name with parentheses following
